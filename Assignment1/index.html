My Journey

Hello! My name is George Danicico and I am a Computer Science Student at Babes Bolyai University in Cluj Napoca, Romania.

During each semester, we had around 5 different classes, and at each we had several assignments in order to learn by applying
what we were taught during the courses. I am grateful because these homeworks were no easy tasks, and I learnt so much by doing
a lot of researching in order to find different solutions to optimise an algorithm, to solve an unexpected bug or how to use git
which is a must have skill for every developer nowadays, since its the most used version control system.

The core concepts that the University taught me, and are also a must have for any kind of programmer are:

  • Object Oriented Programming

    We studied the 4 pillars of the OOP, abstraction, encapsulation, inheritance and polymorphism, alongside MVC pattern
    and a couple of design patterns like Memento, Observer, Visitor.

  • Data Structures and Algorithms

    We studied how are implemented the Abstract Data Types.
